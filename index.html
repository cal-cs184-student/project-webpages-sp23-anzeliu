Computer Graphics Projects

Anze Liu



Project 1: Rasterizer

A rasterizer with triangle drawing, antialiasing with supersampling, transforms in homogeneous coordinates, barycentric interpolation, and texture mapping.



Project 2: MeshEdit

A mesh editor with geometric modeling, triangle edge manipulations via half-edge data structure, and loop subdivision to upsample 3D meshes.



Project 3-1: Path Tracer
Project 3-2: Additional Path Tracer Features

A path tracer with camera ray generation, ray-scene intersection, Bounding Volume Hierarchy acceleration structure, physically based direct and indirect lighting, and adaptive sampling.



Project 4: Cloth Simulation

A cloth simulator with a grid of masses and springs, numerical integration to update vertex positions under the influence of gravitational and spring force, cloth-object collisions, and various shaders in GLSL, e.g. displacement mapping.
